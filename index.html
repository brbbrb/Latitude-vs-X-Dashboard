Weather - Latitude vs. X

The purpose of this project was to analyze the relationship between latitude and various attributes of weather. To accomplish this, weather data was extracted from the OpenWeatherMap API from over 500 random cities.

After cleaning the dataset, Matplotlib was used to plot these relationships across latitudes. Weather attributes included: cloudiness, humidity, max temperature, and wind speed. This site can be used to view each vizualization created as part of the analysis and explanation of any trends or correlations, as well as the original dataset.
